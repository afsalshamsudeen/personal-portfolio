## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/afsalshamsudeen/personal-portfolio.git
```

### Navigate into the project

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
